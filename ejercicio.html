Etiqueta Descripción Atributos
<img> Inserta una imagen en la página web. src, alt, width, height
<map> Se utiliza junto con la etiqueta <area> para crear mapas de imágenes interactivos. name
<area> Se utiliza dentro de la etiqueta <map> para definir áreas sensibles en una imagen, como enlaces o botones. shape, coords, href
<source> Se utiliza dentro de las etiquetas <audio> y <video> para especificar múltiples fuentes de medios, como diferentes formatos de video o audio. src, type
<track> Se utiliza dentro de la etiqueta <video> para especificar pistas de texto (subtítulos) o pistas de audio adicionales. src, kind, srclang
<object> Se utiliza para incrustar objetos multimedia, como flash o contenido de video, en una página web. data, width, height
<param> Se utiliza dentro de la etiqueta <object> para especificar parámetros para el objeto incrustado. name, value
